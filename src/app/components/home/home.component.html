Welcome to the Mission Planner App Plan and visualize robot movements for industrial site inspections

This app allows operators to plan and visualize the robot's movement for industrial site inspections.

App Structure

The app consists of three main pages:

  • Home: This page provides an overview and instructions on how to use the app.
  • Planner: Create a list of coordinates (x/y) for the robot to follow.
  • Player: Playback the created mission to visually verify the planned path.

Getting Started

  1. Clone the repository: git clone {{ githubUrl }}
  2. Navigate to the project directory: cd mission-planner-app
  3. Install dependencies: npm install
  4. Serve the application: ng serve
  5. Open your browser and navigate to: http://localhost:4200

Building the App

  1. Run the build command: ng build --configuration production
  2. The output will be in the dist/ directory.
